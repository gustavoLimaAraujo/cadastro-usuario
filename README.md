# Cadastro de usuário

Este é um projeto simples e funcional de cadastro de usuários desenvolvido para praticar alguns conceitos de CSS aprendidos em aula, e utilizando um pouco de JavaScript puro. O sistema permite que o usuário insira nome, mensagem e gênero, gerando "cards" visuais dinamicamente na tela.

## Funcionalidades

* **Captação de Dados:** Coleta nome, mensagem e gênero através de inputs e selects.
* **Validação Simples:** Verifica se os campos estão vazios antes de prosseguir.
* **Renderização Dinâmica:** Cria elementos HTML via JavaScript sem recarregar a página.
* **Customização por Gênero:** Altera automaticamente o ícone do card com base na escolha (Masculino/Feminino).

## Tecnologias Utilizadas

* **HTML5:** Estruturação da página e formulários.
* **CSS3:** Estilização dos componentes e cards (arquivo `styles.css` referenciado).
* **JavaScript** Lógica de validação e manipulação dinâmica do conteúdo.
